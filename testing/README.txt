Employee Leave Management System
Built with: HTML, CSS, JavaScript + localStorage

How to run:
1. Open index.html in browser
2. Employee Login: ID = emp101, Password = any
3. Admin Login: ID = admin, Password = any
4. Employee can apply leave and view status
5. Admin can approve/reject leaves using Leave ID

Features:
- Login system for Employee + Admin
- Apply leave with form validation  
- View leave status with Pending/Approved/Rejected
- Admin panel to approve/reject requests
- Data saved in browser using localStorage
